##------【为什么选择学linux运维】------
可能是因为喜欢瞎鼓捣电脑吧！从06年第一次接触电脑开始，对日常的拆电脑清灰、组电脑、电脑中毒进不去系统又开始学习装系统，装完系统显卡不工作又开始学装驱动，再到后来的学机械制图后学了cad制图，3dmax建模、视频剪辑。

   14年家装设计的工作发现，我们这里需要这方面的技能不高，全靠一张嘴推销！再加上市场饱和最严重的一年，想挣钱先学会忽悠人！作图都是免费附带的！想做图就拿个笔纸卷尺到新交工的小区测量房屋画cad结构图，再出效果图给客户谈单时看。
当时家装行业的前辈常说：你走在大街上看到穿西服的人，他不是买保险的那就一定是做家装的！人多了就不值钱了！就这样每个月800元工资干了不到1年。对设计行业越离越远！
